Margins, Borders and Padding in CSS

An important concept to understand when working with Cascading Style Sheets (CSS) is the Box Model.

Each piece of content on a webpage is a box can be wrapped within three other boxes. These three other boxes are: padding, border and margin.

  • The content is the actual element that exists in an HTML file, such as a Heading. In HTML this is called an h1 element.

  • When padding is added, it creates transparent space around the content.

  • When a border is added, it creates space around the padding. It is transparent by default, but it can be made visible with styling.

  • A margin creates more space around the border, and like the padding, it is transparent.

The Heading has moved down the page and to the right, because there is now a large margin around it, separating it away from the edges of the browser window.

Now let's add a solid border that is 20 pixels thick.

You can see that the border has taken up space between the content and the margin we created just before.

You'll also notice that the border is very close to the heading. That can be solved with the final component of the Box Model - the padding.

As you can see, the 20 pixels of padding has created space between the Heading (our content) and the border.
